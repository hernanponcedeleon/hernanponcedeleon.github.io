Tools

I'm the maintainer and main developer of the Dat3M tool suite, a collection of tools to analyze concurrent programs running on weak memory models such x86-TSO, ARM or the Linux kernel.

Since 2020, Dartagnan (our reachability analysis tool) participates in SVCOMP.

Related publications:

  3. BMC with Memory Models as Modules
  4. Portability Analysis for Weak Memory Models. PORTHOS: One Tool for all Model
